## Overview

TouchDesigner Advanced is a node-based visual programming platform from Derivative for creating interactive installations, live visuals, projection mapping, real-time 3D graphics, and data-driven visualizations that respond to sensors, audio, video, and network data in real time. It serves the intersection of art, technology, and performance where visual content must respond dynamically to live input rather than playing back pre-rendered media.

Node-based programming connects operators — generators, filters, transforms, compositors — in visual networks that process video, 3D geometry, audio, data, and control signals in real time. The network paradigm makes signal flow visible and enables rapid prototyping of interactive systems. Changes to any node take effect immediately, enabling live experimentation.

Real-time 3D rendering produces GPU-accelerated visuals including particle systems, instanced geometry, physical simulation, and post-processing effects at frame rates suitable for live performance and installation. The rendering pipeline handles multiple output displays, projectors, and LED walls for immersive environments.

Sensor and data integration processes input from cameras, microphones, MIDI controllers, OSC messages, serial devices, web APIs, and other data sources, enabling interactive installations that respond to physical presence, gesture, sound, and environmental conditions.

For artists, designers, and creative technologists who create interactive experiences, live visuals, and immersive installations, TouchDesigner Advanced is the real-time visual platform that bridges creative vision and technical implementation.

## Performance

TouchDesigner balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
